## Troubleshooting

### Console Not Starting

```bash
# Check Python version (3.8+ required)
python3 --version

# Install missing dependencies
pip install -r requirements.txt

# Run with verbose logging
python3 runehall_aggressive_console.py --verbose
```

### Low Performance

```bash
# Increase thread count
# Edit runehall_aggressive_console.py
self.concurrent_threads = 64  # Increase from 32

# Reduce timeout multiplier for faster execution
self.timeout_multiplier = 0.25  # Reduce from 0.5
```

### Missing Output Files

```bash
# Check output directory
ls -la nightfury_aggressive_data/

# Verify write permissions
chmod 755 nightfury_aggressive_data/
```
